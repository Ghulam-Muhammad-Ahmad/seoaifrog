Accessibility

Focus Ring
All interactive elements: 2px solid var(--brand-glow) + 2px offset. Never suppressed.
Color + Label
Color is never the sole differentiator — always paired with Lucide icon or text label.
Gauge ARIA
aria-label="Score: {n} out of 100" on every SVG gauge element.
Form Labels
All fields have associated <label>. Placeholder alone is never sufficient.
Touch Targets
Minimum 44×44px for all interactive elements. Buttons never go below this threshold.
Grid Headers
All AG Grid columns require headerName for screen reader compatibility.
WCAG 2.1 AA
Full compliance target. Amber/orange palette verified for contrast ratios.
Report Images
All images in generated reports must include descriptive alt text strings.
